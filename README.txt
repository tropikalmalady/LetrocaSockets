 -- SERVIDOR
Para executar o servidor é necessário configurar uma base de dados PostgresSQL e fazer o restore dessa base usando o arquivo letroca.backup.
Então basta executar o arquivo server.py no servidor e digitar as informações de acesso a essa base.

-- CLIENTE
Para executar o cliente basta executar o arquivo client.py e digitar o ip do servidor.
Então para jogar basta seguir as intruções do menu.
